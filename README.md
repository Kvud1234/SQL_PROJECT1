# SQL_mini project1

### Retail Sales SQL Analysis 

This is a simple SQL project I created for learning and practicing data analysis.
The dataset contains retail sales information such as date, time, customer details, category, quantity, and total sales.
All analysis was done using PostgreSQL.

### Dataset Information

| Column Name     | Description                                   |
| --------------- | --------------------------------------------- |
| transactions_id | Unique ID for each transaction                |
| sale_date       | Date of sale (MM/DD/YYYY â†’ converted to DATE) |
| sale_time       | Time of sale                                  |
| customer_id     | Unique customer ID                            |
| gender          | Male / Female                                 |
| age             | Age of the customer                           |
| category        | Product category (Clothing, Beauty, etc.)     |
| quantity        | Number of units bought                        |
| price_per_unit  | Price of one unit                             |
| cogs            | Cost of goods sold                            |
| total_sale      | Total sale amount                             |
